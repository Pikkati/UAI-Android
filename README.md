# UAI Android App

Universal AI Android Application

## Description

Android client for the UAI platform, providing enterprise-grade AI capabilities on mobile devices.

## Getting Started

### Prerequisites
- Android SDK 30+
- Gradle
- Java 11+

### Installation

```bash
git clone https://github.com/Pikkati/UAI-Android.git
cd UAI-Android
./gradlew build
```

## Features

- Real-time AI inference
- Offline capability
- Multi-tenant support
- Enterprise authentication
